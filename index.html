TorchSOM documentation

TorchSOM is the official code for paper PAPER ARTICLE, @CONFERENCE @DATE.

It is a PyTorch-based library for training and visualizing Self-Organizing Maps (SOMs), a type of unsupervised learning algorithm used for clustering and dimensionality reduction. PyTorch makes it easy to integrate this package with other deep learning workflows while benefiting from GPU acceleration.
